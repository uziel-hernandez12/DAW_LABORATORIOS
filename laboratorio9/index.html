Posición absoluta con CSS

  • El posicionamiento absoluto afecta el flujo de los elementos HTML ya que el elemento que es afectado por esta regla se elimina del diseño.
  • Al usar el posicionamiento absoluto se tiene que considerar que el elemento se ubicará con respecto al viewport o ventana del navegador, A menos que se declare la posición de su elemento padre. Es decir, tanto para el elemento padre como para el hijo se debe de declarar la propiedad position.
  • Podemos observar que el orden de los elementos en el flujo original en que se ubicaron determinan esta ubicación en la página, es por eso que el primer elemento (caja-1) superpone a todos los demás.
